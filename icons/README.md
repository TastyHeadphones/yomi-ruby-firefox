# Icon Placeholders

This folder includes a template SVG icon.

For production packaging, generate PNG icons with these names:
- `icon16.png`
- `icon32.png`
- `icon48.png`
- `icon128.png`

Suggested workflow:
1. Open `icon-template.svg` in Figma/Illustrator/Inkscape.
2. Export PNGs in the sizes above.
3. (Optional) Add icon references in `manifest.json` under `icons` and `action.default_icon`.
