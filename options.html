Enter your Yahoo! JAPAN API key (App ID) for FuriganaService. You can create one from the Yahoo! JAPAN Developer portal .

How to get the API key

  1. Sign in to Yahoo! JAPAN Developer Network .
  2. Create an app and enable Japanese language API usage.
  3. Copy the generated App ID and paste it above.
